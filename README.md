Photo Browser
============

This is a Photo Browser sample app for iOS 7.

It allows the user to browse popular photos and search for photos. The photos are retrieved from Flickr and Instagram, though more sources can easily be added since the app is using a generic interface.

Requirements
----------

Xcode 5, iOS 7 SDK and Cocoa Pods.
This app was built and tested for iOS 7 beta 6.

How To Use
----------

Clone this repository and run 'pod install'. Then open 'PhotoBrowser.xcworkspace'.
